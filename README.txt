# Rémanence

Journal d’exploration musicale immersif pour festivals.

Application PWA développée avec :

- React
- TypeScript
- Vite
- IndexedDB

Objectif : capturer l'énergie, l'émotion et l'expérience d'un set musical en quelques secondes.

## Concept

Après chaque set :

1. Energie ressentie
2. Focus émotionnel
3. Couleur instinctive
4. Ressenti
5. Apprentissage

L'application génère ensuite une mémoire introspective du festival.

## Stack

React + TypeScript + Vite  
PWA Offline  
Stockage IndexedDB
